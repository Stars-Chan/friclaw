# FriClaw

Friday + Claw - An AI Assistant Project

## 简介

FriClaw 是一个基于 AI 的智能助手项目，旨在提供高效、便捷的人工智能交互体验。

## 项目状态

🚧 项目正在开发中...

## 特性

- [ ] 智能对话
- [ ] 任务自动化
- [ ] 多平台支持

## 快速开始

```bash
# 克隆项目
git clone git@github.com:Stars-Chan/friclaw.git

# 进入项目目录
cd friclaw

# 安装依赖
npm install  # 或 pip install -r requirements.txt

# 运行
npm start  # 或 python main.py
```
